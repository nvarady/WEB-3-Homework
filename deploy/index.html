<html> Content

Structure

We will be using the structure we went over in the last lecture.

<h #> ( h1 - h6 )

The <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.

In HTML 5 you are allowed to have more then one <h1> tag per page.

<p> tag

The <p> tag defines a paragraph. Browsers automatically add some space (margin) before and after each element. The margins can be modified with CSS (with the margin properties).

<span> tag

The <span> tag is used to group inline-elements in a document. The <span> tag provides no visual change by itself. The <span> tag provides a way to add a hook to a part of a text or a part of a document.

This tag will become more useful once we start using css.

<br> tag

The <br> tag inserts a single line break. The <br> tag is an empty tag which means that it has no end tag.

Never use the <br> tag as a design choice. There are a few reasons why you would use the <br> tag ( <address> tag ). You will rarely use the <br> tag in this course.

<hr> tag

The <hr> tag defines a thematic break in an HTML page (e.g. a shift of topic). The <hr> element is used to separate content (or define a change) in an HTML page.

Lists

There are 2 list types that we will use for this class. Each list is made up of at least these 2 components. The <ul> or <ol> and at least one <li>.

<ul> tag

The <ul> tag defines an unordered (bulleted) list.

<ol> tag

The <ol> tag defines an ordered (numbered) list.

<li> tag

The <li> tag defines a list item. The <li> tag is used in ordered lists( <ol> ), unordered lists ( <ul> ).

A <li> maybe be the only child tag of a any list <ul> or <ol> tag

<a> tag

The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

How default links appear in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

Don't worry about the appearance of links we will style them in CSS.

<img> tag

The <img> tag defines an image in an HTML page. The <img> tag has two required attributes: src and alt.

Images are not technically inserted into an HTML page, images are linked to HTML pages. The tag creates a holding space for the referenced image.
